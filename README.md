# 👨‍💼 Employee Management System

A modern **ASP.NET Core MVC** application for managing employee records with SQL Server integration. The system provides a user-friendly interface for viewing and registering employees while demonstrating MVC architecture, Entity Framework Core, and database connectivity.

---

## 🚀 Features

* 📋 View employee records
* ➕ Register new employees
* 🗄️ SQL Server database integration
* 🎨 Responsive and modern UI
* 🏗️ ASP.NET Core MVC architecture
* 🔄 Entity Framework Core support
* 📊 Employee and department management

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, Razor Views
* **Backend:** ASP.NET Core MVC
* **Database:** SQL Server
* **ORM:** Entity Framework Core
* **Language:** C#
* **IDE:** Visual Studio / VS Code

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/Abhishek20042/Employee-Management-System.git
cd Employee-Management-System
```

### Restore Packages

```bash
dotnet restore
```

### Build the Project

```bash
dotnet build
```

### Run the Application

```bash
dotnet run
```

Open your browser and visit:

```
http://localhost:5000
```
